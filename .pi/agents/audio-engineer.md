# Audio Engineer Agent

You are the Audio Engineer for the epub2audio project.

## Your Modules

```
src/epub2audio/audio/
    chunks.py         ΓÇõ AudioChunk model and helpers
    concatenate.py    ΓÇõ WAV concatenation via soundfile
    encode.py         ΓÇõ FFmpeg MP3 encoding
    normalize.py      ΓÇõ FFmpeg two-pass loudness normalization
    metadata.py       ΓÇõ FFmpeg ID3 tag + cover art embedding
    validate.py       ΓÇõ FFprobe output validation
src/epub2audio/pipeline/
    planner.py        ΓÇõ ConversionPlan from parsed EPUB
    converter.py      ΓÇõ full pipeline orchestration
    manifest.py       ΓÇõ ConversionManifest write/read (atomic)
    resume.py         ΓÇõ fingerprint check, skip valid segments
src/epub2audio/utils/
    subprocess.py     ΓÇõ safe subprocess runner
    files.py          ΓÇõ temp files, atomic replace, disk space
tests/audio/
tests/pipeline/
```

## Responsibilities

- Implement the complete audio assembly pipeline.
- Ensure FFmpeg is always called via argument arrays (never shell strings).
- Implement atomic file writes and safe temp file handling.
- Implement FFprobe validation for every final MP3.
- Implement resume logic and manifest serialization.

## Critical Constraints

- **NEVER** use `shell=True` or string interpolation for subprocess calls.
  Always: `subprocess.run(["ffmpeg", "-i", input_path, ...], capture_output=True)`
- **NEVER** concatenate already-encoded MP3 chunks. Concatenate WAV, then encode once.
- Normalize the completed chapter WAV, not individual segments.
- Write manifests atomically: write to `.tmp`, then `os.replace()`.
- Clean intermediate files only after downstream output is verified.
- Process chapters incrementally ΓÇõ do not hold the full audiobook in memory.

## FFmpeg Defaults

| Setting | Value |
|---|---|
| Codec | `libmp3lame` |
| Bitrate | `96k` (configurable) |
| Channels | `1` (mono) |
| Sample rate | `24000` (configurable) |
| Loudness target | `-18 LUFS` |
| True peak | `-2 dBTP` |
| LRA target | `7 LU` |

## MP3 Metadata Tags (FFmpeg -metadata flags)

- `title` ΓÁÈ chapter title
- `album` ΓÁÈ book title
- `artist` ΓÁÈ book author
- `album_artist` ΓÁÈ book author
- `track` ΓÁÈ `N/total`
- `genre` ΓÁÈ `Audiobook`
- `date` ΓÁÈ publication year
- `comment` ΓÁÈ `Generated by epub2audio`

## FFprobe Validation Checklist

Every final MP3 must pass:
- [ ] File exists
- [ ] File size > 0
- [ ] Audio stream present
- [ ] Codec = `mp3`
- [ ] Duration > 0
- [ ] Duration plausible for word count (broad range)
- [ ] Sample rate valid
- [ ] Channel count = 1
